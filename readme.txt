INSTALLATION:
-------------

BINARY:

  WIN: 
  Unzip the files and run the executable.

  OS/X:
  Not released yet.

  LINUX: 
  Not released yet.


SOURCE:

NOTE: Requires that the QT 4.2 development framework is installed and configured 
      correctly. You can download a free Open Source version of QT from Trolltech's
      website: http://www.trolltech.com/

1. Unzip the source files. 
2. cd /path/to/the/source/files/
3. qmake
4. make -or- nmake -or- mingw32-make

1. Introduction

GT-6B Fx FloorBoard is an application that lets you use your personal computer via midi cable to set a variety of GT-6B parameters in a graphic format.

Back to Table of Contents

2. Connections and Settings

1.      Following the instructions in the GT-6B Owner's Manual, connect the GT-6B and your computer via midi cable. You cannot run GT-6B Fx FloorBoard correctly if the GT-6B and the computer are not properly connected.

2.      Turn on the power to the GT-6B.

3.      Then start up GT-6B Fx FloorBoard Editor.

4.      If you are running GT-6B Editor for the first time following installation, select the "Set Up MIDI devices..." from the "Preferences" menu, and select connected computer midi port for "Midi Input/Output", then close the dialog.

5.      If for some reason communications between GT-6B Fx FloorBoard Editor and the GT-6B are interrupted, re-select the "Connect" button from the "staus bar" menu, and then ensure the connect led indication is ‘on’.

********** SPECIAL NOTE!!!!! *********** Due to the editor being not fully completed, an error message may appear, indicating

***********that the GT6B is not connected. *****************

    • Do not disconnect the Midi cable connecting the GT-6B while GT-6B Fx FloorBoard Editor is running.
    • Connect the ‘computer midi output’ to the ‘GT-6B midi input’ and the ‘computer midi input’ to the ‘GT-6B midi output’.

4. Using the GT-6B Fx floorboard Editor

* For details on each parameter and the values that can be set for it’s range, refer to the GT-6B Owner's Manual.

4-1. About the Main Window

The main window is displayed when you start up GT-6B Fx FloorBoard Editor.

4-2. Reading and Writing Settings

            When the EDITOR program is started, the GT6B must be selected to the required patch and  in “BULK LOAD” mode to enable

 reading of parameters. After parameters have transferred, then exit from “BULK LOAD” mode to edit parameters.

The GT-6B's settings are read into GT-6B Editor automatically when you start up GT-6B Editor.

If, in addition to GT-6B Editor, you use the GT-6B in combination with external devices (such as a MIDI sequencer or MIDI foot controller) that will be controlling it, the settings on the GT-6B itself and those shown in the GT-6B Editor screen may differ.
In such instances, you can click the in the top row of the main window to read the GT-6B's selected patch and system settings so that the GT-6B and the GT-6B Editor screen settings match.

Alternatively, you can click the in the top row of the main window to transmit the GT-6B Editor settings to the GT-6B.

When writing patch settings that have been edited with GT-6B Editor to the GT-6B's User patches, don’t click the in the top row of the Edit window to write the data to the selected write-destination patch.

TO WRITE to edited patch to the GT6B memory, press the WRITE button on the GT6B unit itself, and select the destination to WRITE, then press the WRITE button a second time to confirm the WRITING to memory. (this does one patch only)

* You can only write data to the User Bank patches.

4-3. Initializing the Patch Data is not yet working correctly

* You cannot initialize the patch's content unless you have the GT-6B connected.

4-6. Operation with Shared Effects

Turning the Effects On and Off

Click the to the left and right of the effect name in the PREAMP/SPEAKER and EQ areas in the left part of the Main window to turn the effects on and off.
The button is lit when the effect is on.

Click the pedal switches of the compact effects pedals in the right part of the Main window to turn these effects on and off.
The LED at the top of the effect is lit when the effect is on.

Using the Knob Controls

To adjust values with a knob, click the knob and drag it to the right (clockwise) to increase the value, or to the left (counterclockwise) to decrease the value.
If the small size of the knobs makes it difficult to get exact, detailed settings, try clicking the mouse and dragging more slowly. As long as you don't release the mouse button, you can change the values no matter where the mouse is located. In this case, the farther from the center of the knob you drag the mouse cursor, the more precisely you can adjust the settings values.
If only numerals are displayed for the value, you can click that box and then use the computer keyboard to input the value directly.

DETAIL Button

Clicking on the shown as DETAIL, opens a sub-window in which all of that effect's parameters are displayed.
This switches to while the sub-window is open.
You can make detailed settings for each effect in the sub-window.
Click once more to close the sub-window.

5. Troubleshooting

·         When the expression pedal connected to the GT-6B is operated, this is not reflected in GT-6B Editor.

    • Changes in parameter values that have been produced by means of EXP PEDAL/CTL 1,2 or an external MIDI device (such as the FC-200) connected to the GT-6B are not reflected in GT-6B Editor. Press the GT-6B Editor SYNC button when you want the settings of the GT-6B Editor to match those of the GT-6B.

·         The content in all of the User patches is not saved, even when "Save" or "Save As..." is executed.

    • With "Save" or "Save As...," only the selected patch parameters are saved. To save the content of all of the User patches, use a GT-6B Librarian or Bulk Dump software.

·         Some of the characters in the patch name appear differently on the GT-6B and in GT-6B Editor.

    • This is because the characters in the GT-6B differ from those used in GT-6B Editor.
      The [Backslash] displayed in GT-6B Editor appears as the [Yen Symbol] on the GT-6B.
      The "tilde" (~) displayed in GT-6B Editor appears as a space on the GT-6B.

·         When you adjust the GT-6B's knobs, this is not reflected in GT-6B Editor.

·         When you switch patches on the GT-6B, patches in GT-6B Editor do not switch.

·         When you adjust the GT-6B's QUICK FX, this is not reflected in GT-6B Editor.

    • Is the SYSTEM parameter MIDI:KnobCtrlOut set to Off?
      • When this parameter is set to off, various kinds of information, including knob adjustments, switched patches, QUICK FX, and so on are not transmitted from the GT-6B. Set this parameter to On if you need to use such information with GT-6B Editor.

·         When you switch patches in GT-6B Editor, the GT-6B itself does not switch patches.

    • Is any screen other than the Play screen displayed on the GT-6B?
      • Press the GT-6B's EXIT button to bring up the Play screen.
      • Bring up the Play screen on the GT-6B when using GT-6B Editor.

·         Patches cannot be switched with a foot controller or similar control connected to the GT-6B's MIDI IN connector.

    • Are the GT-6B and your computer connected via MIDI?
    • To use extra external MIDI equipment, the MIDI connections must be Daisy Chained, and the Other devices must support MIDI THRU function.
      • When the GT-6B and a computer are connected by means of MIDI, all data arriving at the GT-6B's MIDI IN connector is sent via MIDI to the connected computer, rather than being processed by the GT-6B.

·         Patches displayed on the GT-6B and on the GT-6B Editor do not match.

    • You may have switched the GT-6B's patch on the following occasion.
      • When the GT-6B Editor is reading data from the GT-6B during the Editor's startup.
      • When the GT-6B Editor's READ button is pressed, and the Editor is reading data from the GT-6B.
      • When the GT-6B Editor's SYNC button is pressed, and the Editor is synchronizing data with the GT-6B.

In such cases, the GT-6B Editor cannot properly receive the patch change information from the GT-6B.
Please do not switch patches on the GT-6B when "sending." or "Receiving." message is displayed on the bottom statusbar.

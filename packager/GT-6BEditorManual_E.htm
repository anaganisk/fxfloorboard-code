FxFloorBoard Quick start Help

 

Connecting the Boss GT to a computer: requires 2 midi cables and a midi adapter

                Plug the GT ‘midi-out’ to the midi adapter’s ‘midi-in’

Plug the GT ‘midi-in’ to the midi adapter’s ‘midi-out’

Select the midi adapter which the Boss GT is connected to from the “Preferences/midi

menu, if the midi adapter device is not displayed in the device list, check the midi device

drivers are correctly installed, and that the device is available in the computers system &

hardware device settings (windows = control panel).

 

The Boss GT and the FxFloorBoard program operate in two different modes.

 

Realtime mode- is used for making parameter changes which can be heard in realtime while

‘tweaking and adjusting’.

Bulk Mode- is used for the transfer of patch data to and from the GT, such as reading a patch from

the GT or writing a patch permanently to a selected GT patch memory location.

 

Real-time Editing: Can be done when the program starts and the midi device is connected and

selected. Realtime editing will not permanently overwrite any part of the Boss GT, the data

is sent from the program to the tempory memory area of the GT, and is discarded when the

GT patch is changed. Due to design limitations of the Boss GT, realtime editing will not have

effect when the GT is set to ‘Bulk Mode’.

To access detailed patch settings for each Stomp Pedal, double-click the stomp pedal in an

area which is not a combobox, knob or pedal –switch (i.e on the stomp pedal body).

An edit page window will open, allowing detailed adjustments.

The effects chain can be set by dragging a stomp pedal to a preferred placement.

 

Changing Patch: Open the side bar panel by clicking on the arrow on the left-side of the floorboard

 page. The patch tree will appear, showing User and Preset patches grouped into Banks.

Expand a  Bank  group by clicking on the [+], then expand the selected Bank by clicking

again on the [+] to view a group of Patches. Select the required Patch by clicking on it with a

single mouse click, the Patch on the Boss GT should indicate a change of Patch.

 

Loading Init Patch: On the top status bar is a combobox labeled ‘Init Patches’, a mouse click on this

Will open the combobox and display a selection of patches, click on any patch and it will be

loaded to the editor display only. For the patch to be updated to the GT tempory memory

and make an audible difference, press the ‘write’ button once only.

 

Load/Save Patch File:  A patch file can be loaded or an edited patch saved by selecting the ‘File’

menu. Only single patch files can be used, some custom files may have additional data such

as custom settings or could have corrupted data.  The FxFloorBoard program will advise

you if the loaded file is not correct type. When the file is loaded, the editor display will

update, and the GT data can be updated too by pressing the ‘write’ button once.

 

Re-Name a Patch:  To rename a patch, double-click the mouse on the top statusbar display window

showing the patch name. A re-name dialog window will open, allowing the name to be

changed.

 

BULK MODE: This mode is for data transfer, select the ‘Bulk Mode’ button, if connection is

Successful, the button green light will be on. From here you can double click on a Patch tree

Item to retrieve patch data from the GT. When a Bank group is opened, the names are also

returned from the GT. To write a Patch, select the required patch with a single mouse click

 and press the ‘write’ button twice, the first time will update the tempory memory and the

second press will write the patch to the GT, dialog will appear to confirm you wish to write

data.

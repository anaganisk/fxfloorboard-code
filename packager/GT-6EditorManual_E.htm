1. Introduction

GT-6 Fx FloorBoard is an application that lets you use your personal computer via midi cable to set a variety of GT-6 parameters in a graphic format.

Back to Table of Contents

2. Connections and Settings

1.      Following the instructions in the GT-6 Owner's Manual, connect the GT-6 and your computer via midi cable. You cannot run GT-6 Fx FloorBoard correctly if the GT-6 and the computer are not properly connected.

2.      Turn on the power to the GT-6.

3.      Then start up GT-6 Fx FloorBoard Editor.

4.      If you are running GT-6 Editor for the first time following installation, select the "Set up MIDI devices..." from the "Preferences" menu, and select connected computer's midi port device for "Midi Input/Output", then close the dialog.

5.      5. For patch data transfer, the GT6 unit must be set to "Midi Bulk Receive " for the Editor program to be able to request data from the GT6.

6.      Ensure the correct midi device is selected from the preferences menu, ensure the GT6 unit I.D. is set to "1", and two midi cables are connected, one cable from midi device output > GT6 midi input, and one cable from midi device midi-input > GT6 midi-output.

7.      When these conditions are met, then press the "Bulk Mode " button on the Editor for patch data transfer of single patches.

8.      If for some reason communications between GT-6 Fx FloorBoard Editor and the GT-6 are interrupted, re-select the "Connect" button on the Editor program, and then ensure the connect l.e.d lamp indication is ‘on’ on the Bulk Mode button.

 

    • Do not disconnect the Midi cable connecting the GT-6 while GT-6 Fx FloorBoard Editor is running, as data loss may occure.
    • Connect the ‘computer midi output’ to the ‘GT-6 midi input’ and the ‘computer midi input’ to the ‘GT-6 midi output’.
    • For Realtime tweaking of settings ensure the GT6 & Editor are 'not' in Bulk Mode.
    • In realtime edit mode, the bank tree will change GT6 patches by a single mouse click on the bank/patch tree item.
    • In Bulk Mode, the bank tree will change a patch with a single mouse click & retrieve patch data with a double mouse click.

4. Using the GT-6 Fx floorboard Editor

* For details on each parameter and the values that can be set for its range, refer to the GT-6 Owner's Manual.

4-1. About the Main Window

The main window is displayed when you start up GT-6 Fx FloorBoard Editor.

This is Realtime Editing Mode & GT6 patch settings can be adjusted and auditioned at the same time.

To open or close the Bank-Tree side panel, click on the yellow arrow on the left-hand bar.

4-2. Reading and Writing Settings

            When the EDITOR program is to be "Connected', the GT6 must first be in “BULK LOAD” mode to enable Reading or Writing of patches.

The correct midi device must be selected, the GT6 unit I.D. must be set to "1".

After single patches have transfered, then exit from “BULK LOAD” mode to 'Real-time' edit parameters.

The GT-6's current patch settings are read into GT-6 Editor automatically when you are 'Connected' and 'Double Click' the required patch in the Bank Tree.

If, in addition to GT-6 Editor, you use the GT-6 in combination with external devices (such as a MIDI sequencer or MIDI foot controller) that will be controlling it, the settings on the GT-6 itself and those shown in the GT-6 Editor screen may differ.

Alternatively, to syncronise the editor parameters to the GT6, you can single click the in the top row of the main window to transmit the GT-6 Editor settings to the GT-6 (while in Realtime Edit Mode). The editor data is sent to the GT6 temory buffer and is lost if the GT6 is powered off or the GT6 patch is changed.

When writing patch settings that have been edited with GT-6 Editor to the GT-6's User patches, double click the in the top row of the Edit window to write the data to the selected write-destination patch. (the GT6 must be in "Midi Bulk Mode" to write patches to the internal memory.)

Another proceedure to WRITE the edited patch to the GT6 memory is, press the "WRITE" button on the GT6 unit itself, and select the destination patch location to WRITE to, and then press the WRITE button a second time to confirm the WRITING to memory. (This does one patch per write)

* You can only write data only to the User Bank patches. The Preset patches are read only and can not be over-written.

4-7. EDIT WINDOW

 

To open the Edit Window for any effect, double-click on any stomp pedal within the stomp pedal body area. (i.e. not on the button, combobox or knob items).

From this window, all parameters for an Effect can be modified, if the Editor program and the GT6 are 'not' in bulk mode, the parameters can be adjusted in "real-time", so any changes made in the Edit Window will occur audible changes in the GT6 sound while playing guitar. These changes only affect the GT6 tempory buffer and will be lost if the patch is changed or power is lost. The changes can be saved on the GT6 by pressing "write" on the GT6, or by selecting the editor and GT6 into bulk mode and pressing 'write/sync' on the editor, or the changes can be saved to file in the Editor program by accessing the Menu and selecting "save as".

4-8. CHAIN

You can change the effects' connection sequence by selecting the stomp pedal by holding down the left mouse button and dragging and dropping each effect in the desired position in the sequence. (drag the stomp pedal by holding the mouse left button down while the pointer is on the stomp pedal body area ( not on it's button or knobs).

 

Switching effects ON or OFF.

The l.e.d lamp on the stomp pedal  indicates that an effect is switched on;  indicates the effect is turned off. Click an effect stomp button to toggle it on or off.

* Volume Pedal is on at all time; this is not switched off.

4-9. QUICK FX is not yet implemented

Each effect comes with prepared sample settings called "Quick Settings." When you click   , a list of Quick Settings for all of the effects is displayed. Click the Quick Settings for the desired effect from the list.

* Only one Quick Settings can be selected at a time. Repeat the procedure if you want to make multiple settings.

* You cannot use the QUICK FX feature unless the GT-6 is connected.

5. Troubleshooting

·         When the expression pedal connected to the GT-6 is operated, this is not reflected in GT-6 Editor.

    • Changes in parameter values that have been produced by means of EXP PEDAL/CTL 1,2 or an external MIDI device (such as the FC-200) connected to the GT-6 are not reflected in GT-6 Editor. Press the GT-6 Editor SYNC button when you want the settings of the GT-6 Editor to match those of the GT-6.

·         The content in all of the User patches is not saved, even when "Save" or "Save As..." is executed.

    • With "Save" or "Save As...," only the selected patch parameters are saved. To save the content of all of the User patches, use a GT-6 Librarian or Bulk Dump software.

·         Some of the characters in the patch name appear differently on the GT-6 and in GT-6 Editor.

    • This is because the characters in the GT-6 differ from those used in GT-6 Editor.
      The [Backslash] displayed in GT-6 Editor appears as the [Yen Symbol] on the GT-6.
      The "tilde" (~) displayed in GT-6 Editor appears as a space on the GT-6.

·         When you adjust the GT-6's knobs, this is not reflected in GT-6 Editor.

·         When you switch patches on the GT-6, patches in GT-6 Editor do not switch.

·         When you adjust the GT-6's QUICK FX, this is not reflected in GT-6 Editor.

    • Is the SYSTEM parameter MIDI: KnobCtrlOut set to Off?
      • When this parameter is set to off, various kinds of information, including knob adjustments, switched patches, QUICK FX, and so on are not transmitted from the GT-6. Set this parameter to On if you need to use such information with GT-6 Editor.

·         When you switch patches in GT-6 Editor, the GT-6 itself does not switch patches.

    • Is any screen other than the Play screen displayed on the GT-6?
      • Press the GT-6's EXIT button to bring up the Play screen.
      • Bring up the Play screen on the GT-6 when using GT-6 Editor.

·         Patches cannot be switched with a foot controller or similar control connected to the GT-6's MIDI IN connector.

    • Are the GT-6 and your computer connected via MIDI?
    • To use extra external MIDI equipment, the MIDI connections must be Daisy Chained, and the other devices must support MIDI THRU function.
      • When the GT-6 and a computer are connected by means of MIDI, all data arriving at the GT-6's MIDI IN connector is sent via MIDI to the connected computer, rather than being processed by the GT-6.

·         Patches displayed on the GT-6 and on the GT-6 Editor do not match.

    • You may have switched the GT-6's patch on the following occasion.
      • When the GT-6 Editor is reading data from the GT-6 during the Editor's start-up.
      • When the GT-6 Editor's READ button is pressed, and the Editor is reading data from the GT-6.
      • When the GT-6 Editor's SYNC button is pressed, and the Editor is synchronizing data with the GT-6.

In such cases, the GT-6 Editor cannot properly receive the patch change information from the GT-6.
Please do not switch patches on the GT-6 when "sending." or "Receiving." message is displayed on the bottom status bar.

For Gt-3, Gt-5, Gt-6, & Gt-6B

 

Connecting the Boss GT to a computer: requires 2 midi cables and a computer-to-midi adapter

                Plug the GT ‘midi-out’ to the midi adapter’s ‘midi-in’

Plug the GT ‘midi-in’ to the midi adapter’s ‘midi-out’

**Now start the FxFloorBoard program** after all connections are made

Select the midi adapter which the Boss GT is connected to from the “Preferences/midi

menu, if the midi adapter device is not displayed in the device list, check the midi device

drivers are correctly installed, and that the device is available in the computers system &

hardware device settings ( windows = control panel).

 

The Boss GT and the FxFloorBoard program together operate in two different modes.

 

Realtime mode- is used for making parameter changes which can be heard in realtime while

‘tweaking and adjusting’.

Bulk Mode- is used for the transfer of patch data to and from the GT, such as reading a patch from

the GT or writing a patch permanently to a selected GT patch memory location.

 

***USING 'REAL-TIME MODE':***

Real-time Editing: Can be done when the program starts and the midi device is connected and

selected. Realtime editing will not permanently overwrite any part of the Boss GT, the midi data that

is sent from the program goes to the tempory memory area of the GT, and is discarded when the

GT patch is changed. Due to limitations of the Boss GT firmware, realtime editing will not have

effect when the GT is set to ‘Bulk Mode’.

Bulk Mode is used only for patch transfering therefor 'tweaking and adjusting will have no effect'.

To access detailed patch settings for each Stomp Pedal, double-click the required stomp pedal in an

area which is not a combobox, knob or pedal –switch (i.e on the stomp pedal body).

An edit page window will open, allowing detailed adjustments.

The effects chain order can be set by 'dragging & dropping' a stomp pedal to a preferred place in the effect chain.

 

Changing Patch: Open the side bar panel by clicking on the arrow on the left-side of the floorboard

 page. The patch tree will appear, showing User and Preset patches grouped into Banks.

Expand a  Bank  group by clicking on the [+], then expand the selected Bank by clicking

again on the [+] to view a group of Patches. Select the required Patch by clicking on it with a

single mouse click, the Patch on the Boss GT should indicate a change of Patch.

 

Loading Init Patch: On the top status bar is a combobox labeled ‘Init Patches’, a mouse click on this

Will open the combobox and display a selection of patches, click on any patch and it will be

loaded to the editor display only. For the patch to be updated to the GT temporary memory

and make an audible difference, press the ‘write’ button once only.

 

Load/Save Patch File:  A patch file can be loaded or an edited patch saved by selecting the ‘File’

menu. Only single patch files can be used, some custom files may have additional data such

as custom settings or could have corrupted data.  The FxFloorBoard program will advise

you if the loaded file is not correct type. When the file is loaded, the editor display will

update, and the GT data can be updated too by pressing the ‘Write’ button once.

In real-time mode a patch can be written to a GT patch memory location by pressing

the 'Write' button (twice) on the GT floor unit.

 

Re-Name a Patch:  To rename a patch, double-click the mouse on the top statusbar display window

showing the patch name. A re-name dialog window will open, allowing the name to be

changed.

 

 

**USING 'BULK MODE':***

This mode is for patch data transfer only, select the ‘Bulk Mode’ button, if the connection is

successful, the button green light will be on.

From here you can 'double click' on a Patch tree Item to retrieve patch data from the GT.

When a Bank group is opened, the names are also returned from the GT.

 

**Patch Tree Note ** Currently the program will read each bank folder opened,

starting with the first ones opened, this may take some time for all the midi patch data to be

sent to the program, so patience is required until the status-bar indicates midi activity has finished.

Selecting another bank before this process has finished can cause the program to crash (for the moment).

 

Writing a Patch to GT: To Write a Patch, either a currently tweaked/edited patch or an opened Patch from file,

to the current GT memory location, or another location by selecting the required patch location on the Bank/Patch tree

with a single mouse click,  and thenpress the ‘write/sync’ button twice on the program,

the first press will update the GT temporary memory location and the second press will write the new patch data to the GT,

dialog will appear to confirm your wish to write data permanently to this location.

 

**SWITCHING BETWEEN 'BULK & RealTime MODES':***

The FxFloorBoard program and the Boss GT must both be switched to the same modes together.

The GT unit must be set to 'BULK MIDI RECIEVE MODE' as described in the GT user manual.

The GT unit display will show 'waiting.....' or "idling...'. When leaving "bulk Mode" on the GT unit

by pressing the EXIT button, the GT unit will do data check on any new saved patch data,

this wait may take about 5 seconds.
